# Website Auditor Pro

An enterprise-grade, full-stack web application designed to perform deep UX/UI, SEO, and performance audits on any website.

## Architecture

- **Frontend:** React.js (Custom matte-minimalist UI, Glassmorphism, CSS3)
- **Backend:** Python / Flask
- **Data Integration:** Google PageSpeed Insights API
- **Reporting:** Automated PDF Generation (`html2pdf.js`)

## Features

- **Live Performance Scraping:** Fetches real-time SEO, Accessibility, and Performance scores.
- **Competitor Battle Mode:** Compare two distinct domains side-by-side.
- **Insight Generation:** Automatically isolates and flags top UX/UI bottlenecks.
- **Agency PDF Export:** One-click download of audit reports for client presentations.
